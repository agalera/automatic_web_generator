# AWG
Automatic web generator

web example: https://kianxineki.github.io

Install:

```bash
pip install awg
```

if a settings.json automatically generates the web.
If no settings.json generates several files by default

First use:
```bash
awg
vim settings.json # configure
```

Use:
```bash
awg
```
